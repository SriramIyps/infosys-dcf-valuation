# infosys-dcf-valuation
I built a DCF model to estimate the intrinsic value of Infosys by projecting future free cash flows, discounting them using WACC, and calculating terminal value. The final output gives an estimated share price based on these assumptions.
# DCF Valuation of Infosys

## Overview
This project focuses on estimating the intrinsic value of Infosys using the Discounted Cash Flow (DCF) method. The goal was to understand how future cash flows, discount rates, and growth assumptions impact the valuation of a company.

## Methodology
The DCF model was built using the following steps:

1. Forecasted Free Cash Flows (FCF) for 5 years based on projected growth.
2. Calculated the Weighted Average Cost of Capital (WACC) using the CAPM model.
3. Estimated Terminal Value using both:
   - Perpetuity Growth Method
   - Exit Multiple Method (EV/EBITDA)
4. Discounted all future cash flows to present value.
5. Derived Enterprise Value and Equity Value.
6. Calculated the implied share price.

## Key Assumptions
- Growth Rate: 6%
- WACC: ~10%
- Terminal Growth Rate: 3%
- EV/EBITDA Multiple: 19x

## Results
The model estimates an intrinsic value of approximately ₹1591.57 per share.

## Conclusion
Based on the DCF analysis, Infosys appears to be fairly valued to slightly undervalued under moderate growth assumptions. The valuation is sensitive to changes in growth rate and discount rate.

## Disclaimer
This model is simplified for educational purposes and is not intended for investment decisions.
